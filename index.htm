Quick Note

寫筆記可以有另一種角度

Quick Note 是一款記筆記的 Web 應用程式，可以在手機、平板、筆電、電腦上使用快速便捷的語法，記錄您想記錄的每一刻。

我們的特色


華語輸入

用過筆記app的人都很清楚，很多筆記軟體適合用於英文輸入的介面，而我們的產品將著重在使用中文輸入法的情況下建立格式、排版，完成一樣的功能


隨時隨地

市面的筆記app目前多以筆電、電腦為客戶群；我們想針對手機的使用族群來設計，讓記筆記可以在任何時間、任何地點


雲端儲存

使用雲端資料庫，將檔案遠端儲存，不佔手機空間


我有點子

線上資源庫歡迎使用者分享自己的慣用語法、筆記排版、筆記經驗，一起讓做筆記這件事更輕鬆、更點單、更普及

使用方法

透過打入關鍵字，並在不需切換輸入法的情況下輸入起始與終點標籤(如右動畫中的 ˇ 和。)，再按下↵，就可以輕鬆建立各項排版、標示功能
